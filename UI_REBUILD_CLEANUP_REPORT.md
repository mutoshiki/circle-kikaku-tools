# UI再構成メモ

## 変更方針

スクリーンショットで目立っていた「細いグレー枠で全要素を囲む」見た目をやめ、カードは背景差・余白・薄い影で区切る方向に整理した。

## 主な整理

- `assets/css/04-cars-members-tray.css`
  - 旧仕様の複数車割・名前変更UI向けの重複スタイルを削除
  - 現在使う車割/班割トグル、未割り当てトレイ、編集カードの最小限のCSSに再構成

- `assets/css/08-control-consistency.css`
  - 後追いの上書きブロックを整理し、最終調整レイヤーとして再構成
  - トークン、ボタン、サーフェス、編集ツールバー、精算カード、モーダル、モバイル密度の順に整理
  - グレー枠を弱め、カードの外枠依存を減らした

- `index.html`
  - 精算画面の大きな区切りを `section` 化
  - ツールバーと精算パネルに役割クラスを追加

- `assets/js/templates/settlement-templates.js`
  - 車ごとの精算カードを `article` に変更
  - 合計支払、ガソリン代、諸経費の階層が読みやすい構造に整理
