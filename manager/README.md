# 企画ライブラリ

`circle-kikaku-tools` と同じオリジンに配置し、過去に開いた企画のローカル保存データを一覧管理する静的Webアプリです。

## 配置

リポジトリ直下の `manager/` として配置すると、以下で開けます。

- `https://mutoshiki.github.io/circle-kikaku-tools/manager/`

同一オリジンの `localStorage` から次のキーを読み込みます。

- `sampokai_v10_split_<ROOM_ID>`: 最新の企画データ
- `syawari_history_<ROOM_ID>`: 履歴スナップショット

## 主な機能

- 端末内の企画を自動検出
- 企画名・参加者名・タグ・ROOM IDの横断検索
- 準備中／進行中／完了、お気に入り、保管済みで整理
- 管理用の名前、開催日、タグ、メモを企画本体とは別に保存
- 企画データの複製と新しいROOM IDの発行
- 選択した企画の一括お気に入り・保管・バックアップ
- JSONバックアップの書き出し／読み込み
- モバイルのボトムナビゲーションと大画面のナビゲーションレール
- ライト／ダークモード

## 確認用

ローカルでサンプル表示する場合は `?demo=1` を付けます。

```text
http://localhost:8000/?demo=1
```
