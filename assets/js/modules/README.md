# assets/js/modules

Classic-script helper modules that are loaded by `index.html`.

- `ui.js`: shared alert/confirm/toast/status UI.
- `history.js`: room-scoped history read/write.
- `drag-cards.js`: small drag helpers used by manual card and sheet drag.
- `settlement.js`: tiny settlement helper shim kept for compatibility.

Unused future-only modules were removed in the A cleanup so AI edits do not target files that are not loaded.
