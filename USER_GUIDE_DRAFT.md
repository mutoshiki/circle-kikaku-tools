# 使い方ガイド草本（スマホ版）

参加者登録から車割・班割、共有、精算までをまとめて行える。

[図1：スマホ画面上部。画面切り替え、ロック、共有、メニュー]

## 1. 参加者を登録する

「車割・班割」から「参加者登録」を開く。フォーム回答は見出し行ごと貼り付け、「読み込む」→内容確認→「登録内容で更新」の順。

[図2：スマホの参加者登録画面]

## 2. 車割を作る

車出しにした人ごとに車が作られる。未割り当ての人をドラッグするか、「空席」「ランダム」で自動配置する。固定した人は動かない。

[図3：スマホの車割画面]

## 3. 班割を作る

「班割」に切り替え、班長と班員をドラッグして配置する。定員、自動配置、固定の使い方は車割と同じ。

[図4：スマホの班割画面]

## 4. 共有画面を整える

車割・班割・タイムテーブルが参加者向けに表示される。右下の鉛筆から直し、「完了」で保存する。表は横移動とピンチ拡大に対応。

[図5：スマホの共有画面]

## 5. 予定を入れて共有する

「概要」でメモとタイムテーブルを編集する。タイムテーブルは共有画面にも表示される。

[図6：スマホの概要画面]

内容ができたら、右上の鍵で編集をロックし、共有ボタンからリンクをコピーする。

## 6. 精算する

最初に端数処理などを設定する。車割を使わない企画は「人数だけで精算する」を選ぶ。

[図7：スマホの精算設定画面]

各車の「編集」から距離・燃費・単価・高速代などを入力する。必要ならレンタカー料金も追加する。

[図8：スマホの車ごとの費用入力画面]

移動距離計算ツールからGoogleマップを開き、往復距離を確認する。自宅住所ではなく近くの施設などを使う。

[図9：スマホの移動距離計算ツール]

合計、ドライバーへの支払い、集金状況を確認する。「精算メモをコピー」で連絡用の文章をコピーできる。

[図10：スマホの精算結果上部]

[図11：スマホの支払いチェックと共有用テキスト]

## 7. 保存と復元

変更はこの端末に自動保存される。共有同期中なら、同じリンクを別の端末で開いて続きから使える。履歴やリセットは右上のメニューにある。
