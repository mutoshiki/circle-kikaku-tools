# 配色再設計レポート

## 方針

前回の形状・余白・導線は維持し、青を中心とした配色体系のみを根本から再設計した。

新しい方向性は「温かいフィールドノート」。ライトモードは紙・石・木を思わせる温かいニュートラル、ダークモードは黒ではなくオリーブを含むチャコールを基調とする。

## 色の役割

- 主要操作・選択中: 深いオリーブ
- 割勘: セージ
- 部費: 黄土
- 支払い: ワイン
- 成功: 落ち着いたグリーン
- 注意: アンバー
- 危険・エラー: 赤
- 背景・カード・入力欄: 同じ色温度の段階的なニュートラル

## 主な変更

- ライト／ダークの背景、サーフェス、文字、罫線、影を全面刷新
- 青い主要ボタン、選択タブ、編集ボタン、フォーカスリングをオリーブへ変更
- 支払いの青い意味色をワインへ変更
- 男性・3年生などに残っていた青系バッジをセージ／黄土系へ変更
- ドラッグの戻し先表示に残っていた青をオリーブへ変更
- ブラウザのテーマカラーを新しい背景色へ変更
- `palette-blue`、`ui-blue`など未使用・旧来の色名を削除
- 低レベルパレットを実使用色だけに整理

## 維持したもの

- 画面構造、カード形状、余白、角丸、ボタン配置
- 車割・班割・共有・精算の操作順
- 各状態のラベル、アイコン、選択表示
- 精算3区分の意味と計算ロジック
- ライト／ダーク切替の操作

## 検証

- CSS lint: 合格
- 静的契約: 73件合格
- UI操作テスト: 10件合格
- 詳細監査: 4件合格
- 視覚回帰: 360 / 390 / 430 / 768 / 1280 / 1440px 合格
- 390pxライト／ダーク各35状態: コンソールエラー0、通信失敗0、横スクロール0
