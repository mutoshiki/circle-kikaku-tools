# 共有画面の旧操作移植・参加者登録の三角アイコン復元

## 使用ファイル

- 現在版の基準: `circle-kikaku-tools-direct-pinch-overview-layer-fixed.zip`
- 操作移植元: `circle-kikaku-tools-user-guide-rebuilt(1).zip`
- 出力: `circle-kikaku-tools-legacy-shared-gesture-restored.zip`

## 1. 共有画面の操作

旧ファイルの `assets/js/features/sheet/02-viewport-controls.js` にあった操作方式を、現在版の共有画面へ移植した。

復元した操作は次のとおり。

- 1本指ドラッグで上下左右へ自由に移動
- マウスドラッグで上下左右へ自由に移動
- 2本指ピンチで拡大・縮小
- ピンチ中心を基準に表示位置も追従
- マウスホイール位置を基準に拡大・縮小
- 倍率範囲は旧版と同じ `0.3〜4`
- スマートフォン初期倍率は旧版と同じ範囲を使用

現在版の画面構造・配色・見出し・概要レイヤー・精算機能は維持した。操作対象だけを現在の `#sheet-content` に合わせて移植している。

ズームボタンは復活させていない。

## 2. 根本編集した箇所

- `assets/js/features/sheet/02-viewport-controls.js`
  - スクロール位置を動かす方式を削除
  - 旧版の `sheetX / sheetY / sheetScale` による変換方式へ変更
  - 1本指移動と旧版のピンチ計算を復元
- `assets/css/sheet-view/layout/01-sheet-frame.css`
  - 共有キャンバスをネイティブスクロール所有から、旧版と同じ自由移動用のクリッピング領域へ変更
- `assets/css/sheet-view/gestures/01-touch-navigation.css`
  - `touch-action: none` と変換基準を正規定義
  - 移動中の選択防止とカーソル状態を整理

CSS末尾への追加スキン、`!important`、インライン上書きは使用していない。

## 3. 参加者登録の開閉アイコン

「貼り付け方を見る」「自動判定の仕組み」の左側へ、開閉三角を復元した。

- 閉じているとき: 右向き
- 開いているとき: 下向き
- 48pxのタップ領域は維持
- 2つの説明で同じ実装を共有

変更元は `assets/css/guides-modals/import-guide/01-import-shell.css` の既存 `summary` 定義であり、別ファイルから上書きしていない。

## 4. 変更していないもの

- 共有画面のカード、配色、文字組み
- 「車割」「班割」の文言
- 共有画面の編集ボタン
- 概要を最上位で開くレイヤー構成
- 車の精算画面
- 参加者登録の文章と読み込み処理
- 保存、共有、ドラッグ交換などの既存機能

- 静的契約: 77件合格
- 基本UI操作: 4件合格
- 既存UI契約: 6件合格
- レイアウト回帰: 6件合格
  - 360px / 390pxで自由移動
  - 1本指移動
  - 2本指ピンチ
  - 三角アイコンの表示と開閉
- 視覚回帰: 6画面幅合格

  - 48pxタップ領域

## 6. 実機確認について

Chromiumのタッチイベントで1本指移動と2本指ピンチを確認した。iPhone Safariなど端末固有の慣性や指追従感は、自動試験では完全には再現できないため、最終的な感触だけ実機確認が必要。
